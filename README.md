# Mentor Plaza

Asistente virtual de inversión inmobiliaria de Andrés Plaza (@andresplaza.cl).

## Estructura

- `index.html` — Interfaz del chat (frontend)
- `api/chat.js` — Backend seguro que conecta con Claude API

## Despliegue

Este proyecto está listo para desplegar en Vercel. La API key de Anthropic debe configurarse como variable de entorno:

```
ANTHROPIC_API_KEY=sk-ant-api03-...
```
